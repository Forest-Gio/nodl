# @nodl/nodes

Utility nodes based on the Math-global. Inputs are mapped to native method's arguments.

## Example

```typescript
import { Addition } from '@nodl/math';

const additionNode = new Addition();
```

-   Addition
-   Ceil
-   Cosine
-   Division
-   Floor
-   Logarithm
-   Logarithm2
-   Max
-   Min
-   Modulo

-   Power
-   Round
-   Sign
-   Sine
-   SquareRoot
